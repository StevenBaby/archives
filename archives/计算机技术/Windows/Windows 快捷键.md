| 快捷键 | 功能 |
| - | - |
| CTRL + ALT + DELETE | 调出安全窗口 |
| CTRL + SHIFT + ESC | 打开任务管理器 |
| CTRL + WIN + D | 创建新桌面 |
| CTRL + WIN + ← | 向左切换桌面 |
| CTRL + WIN + → | 创右切换桌面 |
| CTRL + WIN + F4 | 关闭当前桌面 |
