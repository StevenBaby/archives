## 椭球面

$$
\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1
$$

```plot=3d
1 / 8 * x^2 + 1 / 5 * y^2 + 1 / 3 * z^2 = 5
```


## 单叶双曲面

$$
\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1
$$

```plot=3d
x^2 + y^2 - z^2 = 1
```

## 双叶双曲面

$$
\frac{x^2}{a^2} - \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1
$$

```plot=3d
x^2 - y^2 - z^2 = 1
```

## 椭圆抛物面

$$
\frac{x^2}{a^2} + \frac{y^2}{b^2} = z
$$

```plot=3d
x^2 / 5 + y^2 / 3 = z
```

## 椭圆锥面

$$
\frac{x^2}{a^2} + \frac{y^2}{b^2} = z^2
$$

```plot=3d
x^2 / 3 + y^2 / 3 = z^2
```

## 双曲抛物面（马鞍面）

$$
\frac{x^2}{a^2} - \frac{y^2}{b^2} = z
$$

```plot=3d
x^2 / 8 - y^2 / 6 = z
```

## 椭圆柱面

$$
\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1
$$

```plot=3d
x^2 / 8 + y^2 / 6 = 1
```

## 双曲柱面

$$
\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1
$$

```plot=3d
x^2 / 8 - y^2 / 6 = 0.1
```

## 抛物柱面

$$
y = ax^2
$$

```plot=3d
x^2 / 4 = y
```
